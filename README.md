# Words counter

Free simple words counter.

## How to use

1. Open the [website](https://saref111.github.io/words-counter/).
2. Paste your text into the text area.
3. Choose the units you need to count.
3. Click on the button `Count`.

## Features

- Count words
- Count chars
- Count letters
- Count senteces
- Count paragraphs
- Count whitespaces
